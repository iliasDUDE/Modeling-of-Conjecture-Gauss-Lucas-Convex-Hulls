# Modeling-of-Conjecture-Gauss-Lucas-Convex-Hulls
Η εργασία αυτή έγινε στα πλαίσια του μαθήματος Εισαγωγή στο σχεδιασμό και ανάλυση Αλγορίθμων.

Σκοπός της εργασία είναι η μοντελοποίηση μιας εικασίας που περιγράφετε στην αρχή του pdf.

Η εργασία περιλαμβάνει 2 κομμάτια, ένα pdf και ένα αρχείο κώδικα.

Στο αρχείο κώδικα περιλαμβάνεται μία υλοποίηση του προβλήματος, με μία πιο απλή αλλα αργή μέθοδος, όπου παίρνεται ένα τυχαίο πολυώνυμο και κατασκευάζει το G και το G'.

Στο αρχείο pdf γίνεται η θεωρητική ανάλυση του προβλήματος όπου είναι γραμμένοι ψευδοκώδικες για κάθε αλγόριθμο καθώς και οι ιδέες τους για να γίνει η καλύτερη δυνατή ασυμπτωτική ανάλυση του προβλήματος.
